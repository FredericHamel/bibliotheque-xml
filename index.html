Fonctionnement Général

Expérience avec xslt

Le format xslt n'est pas le format le plus simple à la compréhension. Le débogage de xslt n'est vraiment pas facile, car si le XPath est invalide, il n'y a pas de message l'indiquant, la transformationne ne se fait juste pas. En plus, il y le problème avec les espaces de nom, qui, si oubliés, cause un erreur avec le XPath. Dans le commencement, le premier problème était de comprendre comment le xsl:template fonctionne. Finalement, la notion de variables et des a été exploité dans les xslt. Certaines fonctions telles que tokenize et index-of ont été utilisé pour permettre liste les auteurs d'un livre à partir du IDREFS.
