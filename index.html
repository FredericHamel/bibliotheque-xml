Fonctionnement Général

Pour changer la sélection du livre, il faut modifier xsl:param titre dans Bibliotheque.xsl. Pour qu'un livre soit sélectionné, le paramètre doit correspondre à une sous-chaîne de titre. Donc il est possible de sélectionner un livre sans connaître le titre complet. La raison de ce comportement est l'utilisation de la fonction xslt contains.

Dans le cas des auteurs, il y a le xsl:param auteur. Ces paramètres ont un comportement similaire à celui des livres, puisqu'ils utilisent la même fonction contains. Pour valider le xml, nous utilisons un format RNG compacte tel que demandé. Les pays utilisent le format ISO3166-2 de manière exhaustive, puisque la seule autre option que nous avions trouvée était d'ajouter un attribut type au pays représentant le code ISO, ce qui ne convenait pas aux critères d'évaluation, donc nous avons trouvé la liste en xsd, l'avons converti en format RNG compacte et intégré dans notre fichier.

Nous avons choisit une répresentation par liste non ordonnée, puisque qu'il y avait beaucoup de champs qui était variable, tels les auteurs d'un livre, les personnages et tous les champs optionnels. Nous avons donc décidé que la représentation par liste élimine la longueur des lignes qui varient selon le nombre d'éléments(auteurs, personnage, etc.). En plus, la représentation par liste est plus directe.

Expérience avec xslt

Le format xslt n'est pas le format le plus simple à la compréhension. Le débogage de xslt n'est vraiment pas facile, car si le XPath est invalide, il n'y a pas de message l'indiquant, la transformation ne se fait juste pas. En plus, il y le problème avec les espaces de nom, qui, si oubliés, cause un erreur avec le XPath. Dans le commencement, le premier problème était de comprendre comment le xsl:template fonctionne. Finalement, la notion de variables et des a été exploité dans les xslt. Certaines fonctions telles que tokenize et index-of ont été utilisé pour permettre liste les auteurs d'un livre à partir du IDREFS. Le format rnc est assez simple à la compréhension, puisqu'il ressemble beaucoup à une grammaire hors contexte.
